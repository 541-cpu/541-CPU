## 总结

1. 在线搜索到的小标符可以直接粘贴到README中，增加阅读性
2. 使用！[] ()可以将图片直接呈现出来，动图也可以；[]中写图片介绍，（）中粘址，例如![我不管](https://c-ssl.duitang.com/uploads/blog/202404/13/JOS59nv7uWNeJOa.gif)
3. README中尽量避免使用多级标题，让阅读性变差
